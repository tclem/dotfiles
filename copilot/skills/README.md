# Copilot Skills

This directory contains Tim's user-level Copilot skills, synced into `~/.copilot/skills/` by `script/copilot-sync install`.

## Attribution

Some skill-writing practices here are adapted from Jesse Vincent's Superpowers project, especially the `writing-skills` skill:

- Source: https://github.com/obra/superpowers/tree/main/skills/writing-skills
- License: MIT
- Copyright: Copyright (c) 2025 Jesse Vincent

Adapted ideas include trigger-focused skill descriptions, keeping skills reusable rather than project-specific, and pressure-testing process skills against likely agent rationalizations. The skills in this repo are rewritten for Tim's dotfiles, Copilot CLI, and the `copilot/skills/<name>/SKILL.md` layout.
